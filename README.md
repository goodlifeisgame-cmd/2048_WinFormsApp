# 2048 Classic Clone 

## Описание
2048 WinFormsApp – классическая головоломка с динамической сложностью (с возможностью выбора размера поля) и системой сохранения в таблицу лидеров.
Лидеры и прогресс пишутся на диск, чтобы вы могли отслеживать свои победы.

## Техническая часть
* Тип приложения: Десктопная игра для Windows
* C# на .NET 10
* UI-фреймворк: Windows Forms

### Архитектура: 
Разделение на UI-слой (2048WinFormsApp) и библиотеку с бизнес-логикой (2048.Common)

### Анимация
Реализована анимация появление нового числа 

### Система сохранения:
System.Text.Json сериализация данных на диск (текущее поле, счет, рекорд, размер поля)

### Управление:
Клавиатура (стрелки) или мышь

### Особенности реализации:
* Алгоритм сдвига и слияния плиток через итерации по строкам/столбцам
* Генерация новой плитки (2 с вероятностью 90%, 4 — 10%) на случайной свободной ячейке
* Проверка на проигрыш (нет ходов и поле заполнено)
* Таблица лидеров с сортировкой по максимальной цифре
* Изменяемый размер поля (от 3x3 до 8x8) с пересчётом логики
* Изменить размер интерфейса

### Возможные улучшения (на будущее)
- Добавить анимацию перемещения
- Реализовать хранение результатов отдельно для каждой сложности
