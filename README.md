# 2048 Classic Clone 

Простая реализация игры 2048 на C# с использованием WinForms.

Сделал этот проект, чтобы потренироваться в:
- работе с WinForms
- реализации игровой логики
- управлении состоянием приложения

---
## 🎮 Что реализовано

- Полная логика игры 2048
- Управление стрелками с клавиатуры
- Объединение плиток
- Генерация новых чисел
- Подсчёт очков
- Проверка конца игры

## 🧠 Техническая часть
#### 🛠️ Стек

- C#
- .NET
- WinForms

Основная логика завязана на игровом поле (двумерный массив).

- UI (WinForms) отвечает за отображение и ввод
- Логика игры управляет движением, объединением и состоянием поля

На данный момент логика и UI частично связаны между собой — в будущем планирую разделить это более чисто.
